## Project

### IntegriX

AI 기반 통합 보안 분석 플랫폼

- **주요 기술**: Spring Boot, Spring Security, JWT, Redis, MariaDB, MongoDB Atlas, React, TypeScript, AWS EC2, Nginx
- **설명**: 텍스트, 이미지, URL을 분석하여 허위정보, 피싱 위험, AI 생성·합성·조작 의심 이미지, 위험 URL 여부를 확인할 수 있는 서비스입니다.
- **주요 기능**
  - 텍스트 신뢰도 분석
  - 이미지 AI 생성·합성·조작 의심 분석
  - URL 위험도 분석
  - 이메일 인증 기반 회원가입
  - JWT 기반 로그인 및 인증
  - 분석 이력 조회
  - 대시보드 통계
  - 신고 피드백
